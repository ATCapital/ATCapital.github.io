NumPy库使用简介

发表于 2017-02-05 | 分类于 Python

NumPy使用笔记。

实例化ndarray的3个函数方法array，zeros和empty。

numpy.array

创建数组
numpy.array(object, dtype=None, copy=True, order='K', subok=False, ndmin=0)
>>> np.array([1, 2, 3], dtype=complex)
array([ 1.+0.j, 2.+0.j, 3.+0.j])

numpy.zeros

|创建一个用0填充的数组
|numpy.zeros(shape, dtype=float, order='C')
|>>> np.zeros((5,), dtype=np.int)
array([0, 0, 0, 0, 0])

numpy.empty

|生成用随机数填充的数组，速度较快，不推荐使用
|numpy.empty(shape, dtype=float, order='C')
|>>> np.empty([2, 2], dtype=int)
array([[-1073741821, -1067949133], [ 496041986, 19249760]]) #random

numpy.ndarray类

ndarray可以进行向量化运算，例如x **2或者np.sqrt(x)
可以使用x[row][column]查值

常用属性（使用时不加括号）：

Attributes |描述 |例子
T |转置类似.transpose() |x.T
shape |显示数组维度 |x.shape

常用方法：

Methods |描述 |例子
sum(axis, dtype)|按照axis, dtype求和 |
cumsum(axis, dtype)|按照axis, dtype累计求和|
std(axis, dtype)|按照axis, dtype求标准差|

numpy.dtype类

描述数组的存储类型，所有的数据类型是这个类的实例化。详细参数

|数据类型|int8 float_
|使用方式|x = np.float32(1.0)
d = np.dtype(int)
|结构数组|dt = np.dtype([('Name', 'S10'), [('Age', 'i4')])
s = np.array([('Smith', 45), ('Jones', 53)], dtype = dt)

numpy自带sin，使用时注意不要被math覆盖，导入时import math，math.sin/np.sin

numpy.random.standard_normal

|模拟标准正态分布的样本
|numpy.random.standard_normal(size=None)
|s = np.random.standard_normal((4,3))

numpy.log，numpy.log10，numpy.log2，numpy.log1p

|log对数，可以输入array，list。log1p为1plus计算log(1+x)
|numpy.log(x[, out]) = <ufunc 'log'>

numpy.random.randint

|返回随机整数，int。
|numpy.random.randint(low, high=None, size=None, dtype='l')

numpy.allclose

|返回True如果两个数组在一定误差范围内absolute(a - b) <= (atol + rtol * absolute(b))
|numpy.allclose(a, b, rtol=1e-05, atol=1e-08, equal_nan=False)
|np.allclose([1.0, np.nan], [1.0, np.nan], equal_nan=True) # True
