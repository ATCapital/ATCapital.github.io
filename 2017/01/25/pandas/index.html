Pandas库使用简介

发表于 2017-01-25 | 分类于 Python

Pandas使用笔记。参考资料

I/O

pandas.read_csv

|读取csv文件

参数|格式|描述
filepath/buffer|str|./data/table.csv
sep|str, default ‘,’|原始数据间分隔符
header|int|原始数据标题行的位置
index_col|int|原始数据index位置
parse_dates|boolean or int|解析索引列或者其他列的日期
dayfirst|boolean, default False|DD/MM格式用来解析日期格式

内建函数

pandas.date_range

使用pandas.date_range生成DatetimeIndex对象
pandas.date_range(start=None, end=None, periods=None, freq='D', tz=None, normalize=False, name=None, closed=None, **kwargs)[source]

参数|格式|描述
start|string/datetime-like|日期左界
end|string/datetime-like|日期右界
periods|integer/None|期数
freq|string|可以使用5D（5天）更多参数
tz|string/None|时区
normalize|bool|将start，end规划为午夜
name|string|结果索引名称

可以使用df.index将日期赋值给索引
np.array(df)将DateFrame转回ndarray对象

pandas.to_datetime

参考

|将参数转成时间格式
|pandas.to_datetime(*args, **kwargs)
|pd.to_datetime(['2009/07/31', 'asd'], errors='raise')
ValueError: Unknown string format

DataFrame类

|创建dataframe对象。doc
|class pandas.DataFrame(data=None, index=None, columns=None, dtype=None, copy=False)
|df = DataFrame(data=d, index=index)

参数|格式|描述
data|ndarray/list/dict|如果使用dict需要加上index，dict代表colums对应关系。如果是ndarray会自动添加索引值。
index|ndarray/list
columns|ndarray/list

可以使用df['columns'][index]查值。

常用属性（使用时不加括号）：

Attributes |描述 |例子
index |查看/修改index |df.index
values |查看/修改values |df.values
columns |查看/修改columns |df.columns
ix |基于index查看/修改值 |df.ix[['a', 'b']] or df.ix[df.index[1:3]]

常用函数：

Methods|描述|例子
sum(axis)|求和行或列|df.sum(axis=0) # equals df.sum()
cumsum(axis)|累计求和|axis : {index (0), columns (1)}
apply|附加函数|df.apply(numpy.sum, axis=1) # equals df.sum(1)
append|添加数据|先创建新的具有index的dataframe对象
如果没有index：df.append(df2, ignore_index=True)
join(data, how) |合并数据|how默认left向左合并 outer并集
mean(axis)均值|axis : {index (0), columns (1)}
std(axis)|标准差|axis : {index (0), columns (1)}
describe|通用描述|
plot|matplotlib封装器|
max|最大值|axis : {index (0), columns (1)}
size|返回元素数量|
groupby|分组求值，可以多列进行|data.groupby(['col1', 'col2'])['col3'].mean()
corr|求回归系数|method : {‘pearson’, ‘kendall’, ‘spearman’}
fillna|填充缺失NaN数据(np.nan)|method='ffill' # front fill 使用前面的数据填充, 避免前视问题.
rolling(window[, min_periods, freq, center, …])|求移动apply, corr, count, cov, kurt, max, mean, median, min, quantile, skew, std, sum, var|df.rolling(2, win_type='blackman').sum()
比如股价低于前50天最低价入场，current_price < stock.rolling(window=50).min()
shift([periods, freq, axis])|数据向后移动n个index|df/df.shift(1) # 求index i/index i-1

也可以使用NumPy通用函数：np.sqrt(df)
